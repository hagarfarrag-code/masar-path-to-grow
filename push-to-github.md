# Push Commands to Run After Creating GitHub Repository

Once you create the GitHub repository, run these commands:

```bash
# If you need to update the remote URL (replace with your actual repo name)
git remote set-url origin https://github.com/hagarfarrag-code/masar-path-to-grow.git

# Push to GitHub
git push -u origin master

# Or if the default branch is main:
git branch -M main
git push -u origin main
```
